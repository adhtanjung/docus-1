# Quick Start

Get the complete HAER HCM stack running on your local machine in minutes.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js**: v18 or later - [Download](https://nodejs.org/)
- **pnpm**: v8 or later - 
pm install -g pnpm
- **Docker**: For running PostgreSQL - [Download](https://docker.com/)
- **Git**: For version control - [Download](https://git-scm.com/)

---

## Backend Setup (haer-api-core-pf)

The HAER API is built with **NestJS + Fastify + Prisma** and provides a RESTful API for all HCM operations.

::steps

### 1. Clone the Backend Repository

```bash [Terminal]
git clone https://github.com/msa-haer-org/haer-api-core-pf.git
cd haer-api-core-pf
```

### 2. Install Dependencies

```bash [Terminal]
pnpm install
```

### 3. Configure Environment

Copy the example environment file and configure your settings:

```bash [Terminal]
cp .env.example .env
```

Key variables to configure:

| Variable | Description | Default |
| :--- | :--- | :--- |
| DATABASE_URL | PostgreSQL connection string | postgresql://postgres:password@localhost:5432/haer |
| JWT_SECRET | Secret key for JWT tokens | _Required_ |
| PORT | API server port | 3005 |

### 4. Start PostgreSQL Database

Using Docker:

```bash [Terminal]
docker run -d \
  --name haer-postgres \
  -e POSTGRES_USER=postgres \
  -e POSTGRES_PASSWORD=password \
  -e POSTGRES_DB=haer \
  -p 5432:5432 \
  postgres:16-alpine
```

### 5. Run Database Migrations

```bash [Terminal]
npx prisma migrate dev
```

### 6. Seed Initial Data (Optional)

```bash [Terminal]
pnpm run seed
```

### 7. Start the Development Server

```bash [Terminal]
pnpm run start:dev
```

::

::prose-tip
**Backend is running!** Access the Swagger API documentation at: http://localhost:3005/api/docs
::

## Frontend Setup (haer-web)

The HAER Web interface is built with **Nuxt 4 + Vue 3 + TailwindCSS**.

::steps

### 1. Clone the Frontend Repository

```bash [Terminal]
git clone https://github.com/msa-haer-org/haer-web.git
cd haer-web
```

### 2. Install Dependencies

```bash [Terminal]
pnpm install
```

### 3. Configure Environment

```bash [Terminal]
cp .env.example .env
```

Key variables:

| Variable | Description | Default |
| :--- | :--- | :--- |
| NUXT_PUBLIC_API_BASE_URL | Backend API URL | http://localhost:3005 |
| NUXT_PUBLIC_APP_NAME | Application name | HAER HCM |

### 4. Start the Development Server

```bash [Terminal]
pnpm dev
```

::

::prose-tip
**Frontend is running!** Access the dashboard at: http://localhost:3030
::
