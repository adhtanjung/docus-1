# Installation

This guide walks you through the complete installation of the HAER HCM platform.

---

## System Requirements

- **Operating System**: Windows, macOS, or Linux
- **Node.js**: v18.0 or later ([Download](https://nodejs.org/))
- **pnpm**: v8.0 or later (npm install -g pnpm)
- **PostgreSQL**: v14 or later (via Docker or native install)
- **Docker**: v24.0 or later ([Download](https://docker.com/)) - Recommended
- **Memory**: 8GB RAM minimum, 16GB recommended

---

## Database Setup

### Option 1: Docker (Recommended)

Create a docker-compose.yml file:

```yaml [docker-compose.yml]
version: '3.8'
services:
  postgres:
    image: postgres:16-alpine
    environment:
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: password
      POSTGRES_DB: haer
    ports:
      - "5432:5432"
    volumes:
      - haer_postgres_data:/var/lib/postgresql/data

volumes:
  haer_postgres_data:
```

Start the database:

```bash
docker-compose up -d
```

### Option 2: Native Install

**macOS (Homebrew)**:
```bash
brew install postgresql@16
brew services start postgresql@16
createdb haer
```

**Ubuntu/Debian**:
```bash
sudo apt update
sudo apt install postgresql postgresql-contrib
sudo systemctl start postgresql
sudo -u postgres createdb haer
```

## Backend Installation

The HAER API backend provides the core business logic and data management.

### 1. Clone the Repository

```bash
git clone https://github.com/msa-haer-org/haer-api-core-pf.git
cd haer-api-core-pf
```

### 2. Install Dependencies

```bash
pnpm install
```

### 3. Configure Environment

```bash
cp .env.example .env
```

Edit the .env file:

```bash [.env]
DATABASE_URL="postgresql://postgres:password@localhost:5432/haer?schema=public"
JWT_SECRET="your-secure-random-secret-key-at-least-32-chars"
PORT=3005
NODE_ENV=development
SEED_AUDIT_SYSTEM=true
```

### 4. Generate Prisma Client

```bash
npx prisma generate
```

### 5. Run Database Migrations

```bash
npx prisma migrate dev
```

### 6. Seed Initial Data

```bash
pnpm run seed
```

### 7. Start the Server

```bash
pnpm run start:dev
```

::prose-tip
Backend running at http://localhost:3005 | API Docs at http://localhost:3005/api/docs
::

## Frontend Installation

### 1. Clone the Repository

```bash
git clone https://github.com/msa-haer-org/haer-web.git
cd haer-web
```

### 2. Install Dependencies

```bash
pnpm install
```

### 3. Configure Environment

```bash
cp .env.example .env
```

Edit the .env file:

```bash [.env]
NUXT_PUBLIC_API_BASE_URL=http://localhost:3005
NUXT_PUBLIC_APP_NAME="HAER HCM"
```

### 4. Start the Development Server

```bash
pnpm dev
```

::prose-tip
Frontend running at http://localhost:3030
::

## Common Issues

### Database connection failed

Ensure PostgreSQL/Docker is running and DATABASE_URL in .env is correct.

### Port already in use

Change the PORT in .env or kill the existing process.

### Frontend can't connect to API

Verify NUXT_PUBLIC_API_BASE_URL points to the correct backend URL.
